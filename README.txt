This code searches for any occurance of input value in the text file.
If there are more than one occurances with the same lenghth bigger than the input value, the last occurance will be returned
If there is no occurence application will stop

To run the applicatio:
- Please make sure "data" folder is in a same directory as mani.py and search.py
- Using VScode:
    - Run main.py
- Using CMD:
    - cd to the directory that all files are
    - Run main.py
